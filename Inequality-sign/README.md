# Inequality-sign
## 项目名称
-不等号2.0
## 项目简介
-为餐厅、银行等需要排队等候的一系列商家准备,客户可以不用必须到商家店里才能进行排队等候，不用再为去餐厅没有饭桌而忧愁的一款为方便网上排队的APP。
